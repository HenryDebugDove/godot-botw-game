# Godot BotW - 3D动作游戏项目

## 项目简介

Godot BotW是一个基于Godot引擎开发的3D动作游戏项目，灵感来源于《塞尔达传说：旷野之息》。该项目旨在展示Godot引擎在3D游戏开发中的基本功能和最佳实践，包括角色控制、动画系统、物理碰撞等核心游戏机制。

![游戏截图](https://via.placeholder.com/800x450?text=Godot+BotW+Game+Screenshot)

## 功能特性

- **角色控制**：实现了完整的3D角色移动、转向、跳跃等基本动作
- **动画系统**：使用AnimationTree实现流畅的动画过渡和状态管理
- **战斗系统**：支持攻击、防御等战斗动作
- **相机系统**：第三人称视角相机，跟随角色移动
- **物理引擎**：利用Godot的物理系统实现真实的碰撞检测和重力效果
- **资源管理**：包含角色模型、武器、环境等丰富的3D资源

## 系统要求

- Godot Engine 4.3或更高版本
- 支持DirectX 11或Vulkan的显卡
- 至少4GB内存
- 至少2GB可用存储空间

## 安装与运行

### 方法一：使用Godot编辑器打开

1. 下载并安装[Godot Engine 4.3](https://godotengine.org/download)
2. 克隆或下载本项目到本地
3. 启动Godot编辑器
4. 点击"导入"按钮，选择项目文件夹中的`project.godot`文件
5. 点击"运行"按钮启动游戏

### 方法二：使用导出的可执行文件

（注：本项目暂未提供预编译的可执行文件，需要自行导出）

1. 在Godot编辑器中打开项目
2. 点击"项目"菜单，选择"导出"
3. 选择目标平台（Windows、Linux、macOS等）
4. 点击"导出项目"按钮生成可执行文件
5. 运行生成的可执行文件启动游戏

## 玩法说明

### 基本玩法

- **移动**：使用WASD键控制角色移动
- **跳跃**：按下空格键进行跳跃
- **冲刺**：按住Shift键进行冲刺
- **攻击**：点击鼠标左键进行攻击
- **防御**：点击鼠标右键使用盾牌防御
- **视角控制**：移动鼠标控制相机视角

### 游戏机制

- **物理系统**：角色受重力影响，只能在地面上跳跃
- **动画系统**：角色会根据不同的动作状态播放相应的动画
- **碰撞检测**：角色与环境之间有碰撞检测，不会穿过物体

## 技术实现

### 角色控制

- 使用`CharacterBody3D`节点实现角色物理移动
- 通过`Input`类检测玩家输入
- 使用`move_and_slide()`方法处理移动和碰撞

### 动画系统

- 使用`AnimationTree`和`AnimationNodeStateMachine`管理动画状态
- 实现了 idle、running、jumping 等基本动画状态
- 通过代码控制动画状态的切换

### 相机系统

- 使用`Camera3D`节点实现第三人称视角
- 相机跟随角色移动，保持适当距离
- 鼠标控制相机视角旋转

### 战斗系统

- 实现了基本的攻击和防御动作
- 使用`AnimationNodeOneShot`处理一次性动画（如攻击）
- 使用`AnimationNodeBlend2`实现防御动画的过渡
